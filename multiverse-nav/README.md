# Multiverse Navigation System

A sci‑fi themed interactive navigation interface built with HTML, CSS, and JavaScript.  
Features include warp animations, hologram previews, a travel log, and a mini‑map.

## Features
- Full-screen warp tunnel animation
- Dynamic background transitions
- Hologram destination preview
- Travel log with timestamps
- Mini-map showing current location
- Custom backgrounds for 20+ destinations

## How to Run
Just open `navigationsys.html` in any browser.
